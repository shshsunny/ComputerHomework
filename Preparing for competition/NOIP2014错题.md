14. 设变量x 为float 型且已赋值，则以下语句中能将x 中的数值保留到小数点后两位，并将第三位四舍五入的是（ ）。

A. x = (x * 100) + 0.5 / 100.0;

B. x = (x * 100 + 0.5) / 100.0;

C. x = (int) (x * 100 + 0.5) / 100.0;

D. x = (x / 100 + 0.5) * 100.0;

错解：B

正解：C

分析：x*100后再+0.5，是为了四舍五入；用int强制类型转换之后，

可以得到一个以四舍五入方式处理过的.2f格式浮点数*100的整数值，再除以100.0就可以得到其除以100的浮点数了。

18. 设有100 个数据元素，采用折半搜索时，最大比较次数为（ ）。

A. 6 B. 7 C. 8 D. 10

错解：A

正解：B

解析：原理很明白，但是所问是**最大比较次数**，因此要精确计算，避免实际上的误差。
```
100 -> 51 -> 26 -> 13 -> 6 -> 3 -> 2 -> 1，因此共7次比较。
```

阅读程序写结果 4. 
本题目未理解思路（实际上是一个计算素数的程序），附注释后如下：
```cpp
#include <iostream>
using namespace std;
const int SIZE =100;
int main()
{
  int p[SIZE];
  int n, tot, i, cn;
  tot= 0;
  cin>> n;
for(i = 1; i <= n; i++)
p[i] = 1; //p[1]-p[30]中所有元素赋值1
  for(i = 2; i <= n; i++){
    if (p[i] == 1)
      tot++; //计数
    cn = i * 2; //找出2-30中所有2i
    while (cn <= n) {
      p[cn] = 0;
      cn += i; //找出2-30中所有3i
    }
  }//对2-30中素数计数，并输出个数
  cout<< tot << endl;
  return 0;
}
```
输入：30
正解（输出）：10
